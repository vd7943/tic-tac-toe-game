# Tic-Tac-Toe Game

This is a simple Tic-Tac-Toe game built using HTML, CSS, and JavaScript. The game can be played by two players who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game. If all nine squares are filled and neither player has three in a row, the game is declared a draw.

## Features

- Two-player game
- Highlights the winner
- Declares a draw if applicable
- Reset and New Game options
